COP3530: Sliding Puzzle Solver

By Ricky Zhang, Albert Li, and Abbhinav Sriram

Welcome to Sliding Puzzle Solver! This is our final project for COP3530: Data Structures and Algorithms. This project is a web-based application that allows users to compares the abilities of IDA* and Greedy Best-First Search algorithms when solving a 4x4 sliding puzzle.
